# AI-Powered Web Vulnerability Scanner

A web application that scans websites for vulnerabilities using automated tools and AI analysis.

## Features
- Detects OWASP Top 10 vulnerabilities
- AI-powered remediation suggestions (Gemini API)
- Exportable PDF/HTML reports
- Docker containerization

## Tech Stack
- Backend: Python (FastAPI)
- Database: PostgreSQL
- AI: Google Gemini API
- Deployment: Docker + Railway + Vercel
